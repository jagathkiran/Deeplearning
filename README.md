# Deeplearning Projects Repository

## Prerequisites

* Python (3.12)
* pip
* Anaconda

### Setup Virtual Environment
* Using Python

        `python3 -m venv deepl-env`
* Using Conda

        `conda create -n deepl-env`
        `conda activate deepl-env`
### Install Necessary Packages

        `pip3 -r requirements.txt -y`
