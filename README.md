# Deeplearning Projects Repository

## Prerequisites
* pip
* Python (3.11)

**or**
* Anaconda

### Setup Virtual Environment
* **Using Python**
    ```
    python -m venv deepl-env
    ```
* **Using Conda**
    ```
    conda create -n deepl-env python=3.11
    conda activate deepl-env
    ```
    **Deactivate**
    ```
    conda deactivate
    ```
### Install Necessary Packages
```
pip install -r requirements.txt -y
```
